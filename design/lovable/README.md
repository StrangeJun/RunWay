# RunWay Lovable UI Reference

이 폴더는 RunWay Android 앱의 Jetpack Compose 화면 구현을 위한 Lovable UI 참고 자료입니다.

## 목적

- Lovable에서 생성한 모바일 UI 컨셉 보관
- Android 화면 구현 시 디자인, 화면 흐름, 컴포넌트 구조 참고
- 실제 Android 소스코드가 아님

## 폴더 구조

- `source/`: Lovable에서 생성한 웹 UI 소스
- `screenshots/`: 주요 화면 스크린샷

## 구현 시 주의

- Lovable 결과물은 React/Web 기반 참고 자료입니다.
- Android 구현은 `android/` 디렉토리에서 Kotlin + Jetpack Compose로 재구현합니다.
- React/Web 코드를 Android에 직접 복사하지 않습니다.
- 디자인 톤, 화면 배치, 컴포넌트 구조만 참고합니다.
