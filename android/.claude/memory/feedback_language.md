name: 언어 사용 규칙
description: 설명은 한국어로, 기술 용어/파일명/패키지명/API 경로/데이터베이스명은 영어 유지
type: feedback
---

설명과 일반 텍스트는 한국어로 작성하되, 아래 항목은 반드시 영어로 유지한다:
- 기술 용어 (e.g., primary key, endpoint, dependency injection)
- 파일명 및 경로 (e.g., CLAUDE.md, build.gradle)
- 패키지명 (e.g., com.runway.android)
- API 경로 (e.g., /api/v1/courses)
- 데이터베이스명 및 컬럼명 (e.g., PostGIS, course_attempts)

**Why:** 사용자가 직접 요청한 선호 방식.
**How to apply:** 모든 응답에 일관되게 적용. 코드 블록 내부는 원래부터 영어이므로 그대로 유지.
